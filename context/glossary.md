# Glossary
- **Demand Item**: A required headcount/role/shift at a site (OU).
- **OU (Org Unit)**: Grouping aligned to terminals/segments; can be grouped cross-OU for rostering.
- **Product Type / Rank**: APO/AVSO/CVSO etc.
- **Rotation Pattern**: e.g., `111100` (Day=1, Night=2, Off=0).
- **PDL**: Provisional/temporary license override with validity.
- **Delta Solve**: Partial re-optimization for new joiners/changes without resetting published roster.
- **Hard/Soft Constraint**: Must-haves vs optimization goals.
